# Blinkit Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit's sales performance using interactive visualizations.

## Features
- Total Sales KPI
- Total Orders
- Average Sales
- Sales by Category
- Outlet Performance
- Interactive Filters

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Dataset
Blinkit Sales Dataset
